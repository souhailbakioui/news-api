## Exigences
- Laravel 10
- PHP 8.1.6
- phpMyAdmin 5.2.0 

## Utilisation
1. Décompressez le fichier ZIP sur votre machine.
2. Exécutez `php artisan serve` pour démarrer le serveur de développement.
3. Accédez à l'application via votre navigateur à l'adresse http://localhost:8000.
